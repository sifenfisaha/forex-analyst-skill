# 07 — Confluence Scorecard (the trade gate)

Every setup must pass through this scorecard before a plan is issued. It converts "this looks good" into a disciplined, repeatable grade. **No grade ≥ B → NO-TRADE.**

## How to score

Award **1 point** per confluence that is genuinely present (not forced). Be strict — when in doubt, it's a NO. Then map the total to a grade.

### A. Non-negotiables (ALL four required — if any is missing, it's an automatic NO-TRADE, regardless of points)

- [ ] **HTF bias is clear** (or price is at a clean HTF extreme for a reversal). Not mid-range chop.
- [ ] **Liquidity was swept** — a real raid of a defined pool (Asian range / PDH-PDL / equal highs-lows), closing back inside.
- [ ] **MSS / CHoCH confirmed** on the execution TF *after* the sweep (ideally with displacement + FVG).
- [ ] **RR ≥ 1:3** to a real liquidity target, with a logical stop beyond invalidation.

> If those four aren't all true, stop here. **NO-TRADE.** Don't score the rest.

### B. Confluence points (score these once the four gates pass)

- [ ] **+1** Entry sits in an **FVG + OB confluence** (not just one array).
- [ ] **+1** Price is in **discount (for longs) / premium (for shorts)**, ideally in the **OTE 62–79%** zone.
- [ ] **+1** Setup matures inside a **killzone** (London KZ or NY AM KZ).
- [ ] **+1** **Displacement** present (strong, decisive candles leaving FVGs) — momentum confirms intent.
- [ ] **+1** A clean, **named draw on liquidity** as the target (you can point to the exact pool ≥3R away).
- [ ] **+1** **DXY / correlated pair agrees** with the direction.
- [ ] **+1** **No high-impact news** in the entry window.
- [ ] **+1** **Day-of-week / timing** favorable (Tue–Wed expansion, or a textbook Judas reversal).
- [ ] **+1** **Multi-timeframe alignment** (HTF bias, MTF structure, LTF trigger all point the same way).
- [ ] **+1** **Clean chart** — obvious structure, not messy overlapping consolidation.

## Grading

| Points (Section B) | Grade | Action |
|---|---|---|
| **7–10** | **A+** | Full setup. Risk up to **1%**. This is what you wait all day for. |
| **5–6** | **B** | Valid but not premium. Risk **0.5%**, or tighten criteria. Tradeable. |
| **3–4** | **C** | Marginal. **NO-TRADE** — set an alert, wait for it to improve. |
| **0–2** | **D** | **NO-TRADE.** Walk away. |

*(Remember: Section B is only scored if all four Section-A gates passed. Failing a gate = NO-TRADE no matter how many B points exist.)*

---

## Why a hard gate matters

The scorecard exists to stop the two account-killers: **forcing trades** (boredom/FOMO) and **lowering standards after a loss** (revenge). When the user pushes for a trade that scores C, the disciplined answer is the valuable one: *"This is a C — here's the one thing that would turn it into a B, and what to set an alert on."*

Quality over quantity. **A few A+ trades per week beats ten mediocre ones.** The scorecard is how you enforce that on yourself when emotion says otherwise.

---

## Quick verbal version (when you don't need the full checklist)

> "Swept? Shifted? In a good array, in discount/premium? In a killzone? News clear? Target ≥3R to real liquidity? — How many yeses?"

Five+ clean yeses including all four gates = trade it. Fewer = wait.
